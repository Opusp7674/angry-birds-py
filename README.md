Angry Birds in Python
=====================
[Youtube Video](https://www.youtube.com/watch?v=tzXUlzCKvec)

![Alt text](/resources/images/angry-birds-image.png?raw=true "angry-birds")

Requirements
------------
```bash
pip install pygame
```
```bash
pip install pymunk
```

To play
-------
1. Clone the repo.
2. Open the src folder.
3. `python game.py`
